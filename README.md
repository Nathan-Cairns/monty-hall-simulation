## RUN

Just run the cli.py script.

`python3 cli.py`

or 

`./cli.py`

Alternatively run the install.sh script

`sudo ./install.sh`

and then call from the command line as

`mhall`

use the -m flag to run multiple simulations

```shell
mhall -m
python3 cli.py -m
./cli.py -m
```
